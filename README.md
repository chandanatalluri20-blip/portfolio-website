# Developer Portfolio

A personal portfolio website designed to showcase my skills, projects, achievements, and professional profile as a Computer Science student.

## Features

### Home

A professional introduction and overview of my profile.

### About Me

Information about my education, technical skills, interests, and career goals.

### Resume

View and download my latest resume.

### Projects

Interactive project showcase featuring:

* Simple Calculator
* Quiz Generator
* To-Do List
* Weather App

All projects are clickable and can be explored directly.

### Project Highlights

A dedicated section explaining:

* What I learned from each project
* Technologies used
* Key concepts implemented
* Skills gained during development

### Contact

Direct access to:

* Phone Number
* Email Address
* LinkedIn Profile
* GitHub Profile
